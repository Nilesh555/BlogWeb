# Django Blog Web Application

A modern blog web application built with Django that allows users to create, manage, and share blog posts. The platform provides a clean user interface, secure authentication, and efficient content management.

## Features

- User Registration and Authentication
- Create, Update, and Delete Blog Posts
- View All Blog Posts
- Individual Blog Detail Pages
- User Profile Management
- Responsive Design
- Admin Dashboard for Content Management
- Search Functionality
- Secure User Authentication
- Database Integration with Django ORM

## Technologies Used

### Backend
- Python
- Django
- SQLite (Development)
- Django ORM

### Frontend
- HTML5
- CSS3
- Bootstrap
- JavaScript
